# Little Book Haven Management System

A Java-based console application developed using Object-Oriented Programming (OOP) principles.  
This system simulates a bookstore management environment with role-based access and MySQL database integration.

---

## Technologies Used

- Java (OOP Concepts)
- MySQL
- JDBC (Java Database Connectivity)
- NetBeans IDE

---

## Features

- User role-based system (Admin / Customer)
- Add, update, delete books (Admin)
- View available books
- Purchase books (Customer)
- MySQL database integration
- Console-based interaction

---

## OOP Concepts Implemented

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Class & Object design
- Separation of concerns
